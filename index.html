- MOUSE & KEYBOARD -

[ camera ]
alt + LMB: rotate
alt + MMB: pan
alt + RMB: zoom

[ terrain ]
ctrl + LMB: add
ctrl + RMB: remove

[ water ]
space + LMB: source
space + MMB: disturb
space + RMB: sink

[ while adding objects ]
hold R + move mouse: rotate

[ first dynamic object ]
W, A, S, D: push

- AUGMENTED REALITY -

You will need printed AR markers and a good webcam if you wish to try out augmented reality controls
